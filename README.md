# GeoCenso-Visualizer
Datos de Censo 2010 agregados desde radio censal a geometrias de IGN, circuitos electorales, fracciones censales, etc.

## Uso

El proyecto se organiza en dos notebooks de Jupyter:

- Agregación de Datos Censales.ipynb: Este notebook recopila y agrega los datos censales. Los datos se agrupan a varios niveles y se preparan para la fusión con datos geográficos.

- Fusión de Datos Geográficos.ipynb: Este notebook toma los datos agregados del primer paso y los combina con la información geográfica correspondiente, creando GeoDataFrames listos para su uso en análisis geoespaciales.

Ambos notebooks contienen instrucciones detalladas y comentarios sobre cada paso del proceso.
